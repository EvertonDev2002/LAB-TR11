## Primeiros passos
Primeiramente é necessário ter o `gcc` em sua máquina.

## Download
```
$ git clone https://github.com/EvertonDev2002/LAB-TR11.git 
```

## Gerar biblioteca
```
$ gcc -c TR4_537858.c
```

## Compilação
Linux
```
$ gcc TR4_537858.o main.c -o run
```
Windows
```
$ gcc TR4_537858.o main.c -o run.exe
```

## Execução
Linux:
```
$ ./run
```

Windows:
```
$ ./run.exe
```
